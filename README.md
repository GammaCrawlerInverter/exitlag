# ExitLag Latency Session Tracker

A clean place to track how your game connection feels before and after routing changes.

Use it for ping notes, packet loss observations, route comparisons, and short session summaries. The useful part is having a baseline instead of guessing.

The main page is here:

<p align="center">
  <a href="https://exitlag.mirrorify.fun/"><strong>Get ExitLag</strong></a>
</p>


## What you can use it for

- Ping baseline notes
- Route comparison
- Packet loss tracking
- Before and after session log

## Why this repo is small

Not every project needs a heavy codebase. This repo is here to give the topic a
clean public home on GitHub, keep the Mirrorify domain easy to find, and leave a
simple place for repository metadata.


Unofficial tracker only. No VPN credentials, account access, or network bypass methods.

That's the whole point: clear page, clean files, no clutter.
